# 👥 Управление пользователями - Инструкция

## ✅ Что сделано

Создана страница **`/admin/users`** для управления всеми пользователями системы.

## 📋 Что показывается

### Статистика (вверху страницы):
- 👥 **Всего пользователей** - общее количество
- 🎓 **Студентов** - количество студентов
- 👨‍🏫 **Учителей** - количество учителей

### Список пользователей:
Каждый пользователь показывается с информацией:

**Студенты:**
- Имя
- Email
- Группа (ИС-21-1, ИС-21-2, и т.д.)
- GPA (средний балл)
- Дата регистрации
- Кнопка удаления

**Учителя:**
- Имя
- Email
- Предмет
- Дата регистрации
- Кнопка удаления

## 🎯 Фильтры

Три кнопки для фильтрации:
- **👥 Все** - показать всех (студенты + учителя)
- **🎓 Студенты** - только студенты
- **👨‍🏫 Учителя** - только учителя

## 🚀 Как использовать

### Шаг 1: Создать таблицы в Supabase

1. Откройте Supabase Dashboard
2. Перейдите в **SQL Editor**
3. Скопируйте содержимое файла **`create-users-tables.sql`**
4. Вставьте в SQL Editor
5. Нажмите **Run** (или F5)

Это создаст таблицы `students` и `teachers` с правильными настройками.

### Шаг 2: Открыть страницу

1. Войдите как **администратор** (admin@narxoz.kz)
2. В меню выберите **"Пользователи"**
3. Или откройте напрямую: `http://localhost:4004/admin/users`

### Шаг 3: Проверить

Вы увидите:
- Статистику пользователей
- Список всех студентов и учителей
- Возможность фильтровать по роли
- Кнопки удаления

## 🔄 Real-time обновления

Страница автоматически обновляется когда:
- Регистрируется новый студент
- Регистрируется новый учитель
- Удаляется пользователь
- Изменяются данные

**Не нужно обновлять страницу вручную!**

## 🗑️ Удаление пользователя

1. Найдите пользователя в списке
2. Нажмите кнопку **"🗑️ Удалить"**
3. Подтвердите удаление
4. Пользователь будет удалён из таблицы

**Примечание:** Это удаляет запись из таблицы students/teachers, но не из auth.users.

## 📝 Как добавляются пользователи

### Студенты:
1. Регистрация через `/register`
2. Выбор роли "Студент"
3. Выбор группы (ИС-21-1, ИС-21-2, и т.д.)
4. Автоматически создаётся запись в таблице `students`

### Учителя:
1. Регистрация через `/register`
2. Выбор роли "Учитель"
3. Автоматически создаётся запись в таблице `teachers`

## 🎨 Дизайн

- Современные градиентные карточки
- Цветные бейджи для ролей (зелёный для студентов, фиолетовый для учителей)
- Адаптивная таблица
- Плавные анимации

## ⚠️ Важно

1. **Сначала выполните SQL скрипт** `create-users-tables.sql` в Supabase
2. Убедитесь, что у вас есть права администратора
3. Проверьте, что в `.env.local` правильные ключи Supabase

## 🧪 Тестирование

1. Зарегистрируйте тестового студента
2. Зарегистрируйте тестового учителя
3. Войдите как админ
4. Откройте `/admin/users`
5. Проверьте, что оба пользователя видны
6. Попробуйте фильтры
7. Попробуйте удалить тестового пользователя
