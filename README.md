# Система управления колледжем

Веб-приложение для управления расписанием и оценками студентов колледжа.

## Функционал

### Роли пользователей

1. **Администратор**
   - Управление расписанием, аудиториями, учителями и студентами
   - Управление типами оценок
   - Просмотр всех GPA студентов

2. **Учитель**
   - Просмотр своего расписания
   - Добавление и редактирование оценок студентов
   - Фильтрация студентов по группе и дисциплине

3. **Студент**
   - Просмотр расписания своей группы
   - Просмотр своих оценок и GPA
   - Информация о преподавателях и аудиториях

## Технологический стек

- **Frontend**: Next.js 14, React, TypeScript
- **Backend**: Supabase (PostgreSQL, Auth)
- **UI**: TailwindCSS
- **База данных**: PostgreSQL через Supabase

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
```bash
npm install
```

3. Создайте проект в Supabase (https://supabase.com)

4. Выполните SQL-скрипт из файла `supabase-schema.sql` в SQL Editor вашего проекта Supabase

5. Скопируйте `.env.local.example` в `.env.local` и заполните переменные:
```
NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
```

6. Запустите проект:
```bash
npm run dev
```

7. Откройте http://localhost:3000

## Структура базы данных

- `users` - пользователи системы
- `groups` - учебные группы
- `teachers` - преподаватели
- `subjects` - предметы
- `auditories` - аудитории
- `schedule` - расписание занятий
- `grades` - оценки студентов

## Типы оценок

- Лекция (lecture)
- СРСП (srsp)
- СРС (srs)
- Сессия (session)

## Безопасность

Проект использует Row Level Security (RLS) политики Supabase для разграничения доступа:
- Студенты видят только свои данные
- Учителя управляют оценками по своим предметам
- Администраторы имеют полный доступ

## Разработка

Для запуска в режиме разработки:
```bash
npm run dev
```

Для сборки production версии:
```bash
npm run build
npm start
```
