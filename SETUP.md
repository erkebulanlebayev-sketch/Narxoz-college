# Инструкция по настройке базы данных

## Шаг 1: Откройте Supabase Dashboard

1. Перейдите на https://supabase.com/dashboard
2. Войдите в свой аккаунт
3. Выберите проект: **zpycymatynwdbbvgsjab**

## Шаг 2: Выполните SQL-скрипт

1. В левом меню нажмите **SQL Editor**
2. Нажмите **New Query**
3. Скопируйте весь код из файла `supabase-setup.sql`
4. Вставьте в редактор
5. Нажмите **Run** (или Ctrl+Enter)

## Шаг 3: Проверьте создание таблиц

1. В левом меню нажмите **Table Editor**
2. Вы должны увидеть таблицы:
   - groups (4 группы)
   - users
   - teachers
   - subjects
   - auditories (5 аудиторий)
   - schedule
   - grades

## Шаг 4: Запустите приложение

```cmd
npm run dev
```

Откройте http://localhost:4000

## Шаг 5: Зарегистрируйтесь

1. Перейдите на страницу регистрации
2. Заполните форму:
   - Имя: Ваше имя
   - Email: ваш email
   - Пароль: минимум 6 символов
   - Роль: Студент или Учитель
3. Нажмите "Зарегистрироваться"

## Готово!

Теперь вы можете войти в систему и использовать приложение.
